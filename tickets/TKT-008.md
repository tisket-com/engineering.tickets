# TKT-008: Add ticket dependencies (blocks/blocked-by)

## Overview
Allow tickets to declare dependencies on other tickets.

## Requirements
- Add `blockedBy?: string[]` field (array of ticket IDs)
- Add `blocks?: string[]` field (derived/computed)
- Visualize dependency chains
- Warn when trying to close a ticket that blocks open tickets
- Show blocked status in ticket list

- Parse dependency fields from frontmatter
- Build dependency graph when loading tickets
- Add visual indicators for blocked tickets

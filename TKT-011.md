# TKT-011: Link commits and PRs to tickets automatically

## Overview
Automatically detect and display related code changes for tickets.

## Requirements
- Parse ticket IDs (TKT-123) from commit messages
- Show linked commits on ticket detail page
- Optionally auto-update ticket status on merge
- Support GitHub PR integration

## Implementation Notes
- Scan commit messages during git operations
- Store/index ticket-commit relationships
- Add "Related Changes" section to ticket detail
- Consider GitHub webhook for PR status updates
